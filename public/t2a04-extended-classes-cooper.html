W3 Schools Extended Classes: good refrence for extended classes
extending classes are different function you can do to a class, like alter whats in them, add new class items into them, and add other classes ontop of classes
example code
class Car {
constructor(brand) {
this.carname = brand;
}
present() {
return 'I have a ' + this.carname;
}
}
class Model extends Car {
constructor(brand, mod) {
super(brand);
this.model = mod;
}
show() {
return this.present() + ', it is a ' + this.model;
}
}
mycar = new Model("Ford", "Mustang");
document.getElementById("demo").innerHTML = mycar.show();
